Trafic interrompu !


Nombre de tweets par semaine indiquant « trafic perturbé » ou « trafic interrompu » sur les comptes des lignes de transport d’Île-de-France (Métro, RER et Transilien), sur la période du 06/12/2017 au 26/11/2018.

Retrouvez le code sur notre  GitHub : https://github.com/dreamRs/tweets-transports

A propos :

  • Les tweets ont été collectés avec le package rtweet.
  • Les pictogrammes des lignes viennent de la plateforme Open data de la RATP.
  • Les graphiques ont été réalisés avec le package billboarder.
